# mvp-activa-40
Plataforma web HealthTech que conecta usuarios +40 con entrenadores mediante triage inteligente, rutinas personalizadas y seguimiento de adherencia.
Bienvenido a Activa+40 , un proyecto nacido para resolver un problema real: El abandono masivo del ejercicio en personas mayores de 40 años. 

## ¿Cómo funciona?

1. **Triage de Seguridad:** Al registrarse, el usuario declara sus dolencias usando una escala de dolor (EVA). El sistema **bloquea automáticamente** cualquier ejercicio que pueda lastimarlo.
2. **Match Inteligente:** El usuario es emparejado con un *Coach* certificado real, experto en sus necesidades.
3. **Entrenamiento y Feedback:** El usuario entrena y dice qué tan difícil sintió la rutina (RPE). 
4. **Copiloto para el Coach:** El entrenador tiene un panel que procesa ese esfuerzo y le sugiere exactamente cómo ajustar los pesos para la próxima semana.
5. **Rescate Automático:** Si un usuario desaparece por 3, 7 o 14 días, el sistema le avisa al entrenador para que intervenga de forma humana y evite el abandono.

## Stack Tecnológico

Este MVP está construido priorizando la agilidad, la seguridad y una experiencia de usuario fluida sin sobrecargar el navegador:

* **Backend:** Python + Django.
* **Frontend:** HTML5 + Tailwind CSS + HTMX
* **Base de Datos:** SQLite.

## El objetivo del proyecto 

Demostrar que la tecnología no debe reemplazar el contacto humano, sino facilitarlo. Activa+40 usa algoritmos para la seguridad física, pero deja la empatía y la retención en manos de profesionales reales.
